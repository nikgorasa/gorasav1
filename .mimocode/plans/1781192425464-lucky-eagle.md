## Architecture


### Step 9: Create AI Logic

**File:** `gorasa-next/src/lib/ai/holidayPlanner.ts`

**Functions:**
```typescript
export async function generateHolidayResponse(
  messages: Message[],
  context?: { userId?: string; previousItinerary?: Itinerary }
): Promise<{ message: string; itinerary?: Itinerary; quickReplies: string[]; state: string }>
```

**Logic (MVP - rule-based first, upgrade to Gemini later):**
1. Count user messages to determine conversation state
2. Based on state, ask next question or generate itinerary
3. Return message + quick replies + itinerary if generated

**Conversation States:**
- 0 messages → greeting + destination quick replies
- 1 message (destination) → ask duration
- 2 messages (duration) → ask travelers
- 3 messages (travelers) → ask budget
- 4 messages (budget) → generate itinerary
